# Dossier Assets

Ce dossier contient les textures utilisées dans le projet 3D Volleyball Court.

## Ajout de votre texture sand.png

1. Placez votre fichier `sand.png` dans ce dossier (`public/assets/`)
2. Le projet chargera automatiquement la texture au démarrage
3. Si la texture ne se charge pas, une couleur de fallback sera utilisée

## Formats supportés

- PNG (recommandé)
- JPG
- WebP

## Taille recommandée

Pour de meilleures performances, utilisez une texture de 512x512 pixels ou 1024x1024 pixels.
